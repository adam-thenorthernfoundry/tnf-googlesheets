== Description ==
This plugin can be used to upload any Google Sheet to create Posts, Custom Post Types, Taxonomies, Categories and Custom Post Meta. 

This plugin removes the need for a seperate CSV file upload and is ideal for teams working on large data sets in \'google sheets\'.

Easy to map columns from your google sheet with Taxonomies, Categories and Custom Fields.

== Installation ==
1. Upload the Import Posts from Google Sheets plugin to your site
2. Enter your Google Sheets Share link
3. Map Your fields
4. Import!
